# 💜 JulyVerse

**JulyVerse** es una aplicación desarrollada con **Python** y **Streamlit** para llevar un registro personalizado de **KDramas** y **Anime**, utilizando la API de TMDB.

## ✨ Características actuales

- 🎬 Catálogo de KDramas
- 🔍 Búsqueda de series
- ✅ Marcar como visto
- 💾 Almacenamiento local

## 🚀 Próximas características

- 🌸 Catálogo de Anime
- ❤️ Favoritos
- ⭐ Calificaciones
- 📝 Comentarios personales
- 📊 Estadísticas
- 🎨 Nueva interfaz inspirada en BTS

## 🛠 Tecnologías

- Python
- Streamlit
- TMDB API
- Pandas
- SQLite *(próximamente)*
